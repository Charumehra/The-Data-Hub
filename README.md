## Blog API with Authentication

A RESTful API built using Node.js, Express, and MongoDB that allows users to perform CRUD operations on posts along with a basic authentication system.

## Features
 User Authentication (Mock JWT)
 Create, Read, Update, Delete (CRUD) Posts
 MVC Architecture
 RESTful API Design
 MongoDB Database Integration
 Request Logging Middleware
 
## Tech Stack
Backend: Node.js, Express.js
Database: MongoDB, Mongoose
Environment: dotenv
Other: DNS handling for MongoDB SRV connections
